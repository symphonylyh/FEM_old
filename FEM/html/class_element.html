◆ getShape()

virtual Shape* const Element::getShape ( ) const
pure virtual

Get the shape object of this element.

Returns
A pointer to the shape object of this element.
Note
For different element, we would have different shape property, so we return a generic polymorphed Shape class pointer to achieve the standardized behavior of our program. It should also be noted that this is a virtual method, and the Shape property of each element type is stored as a static member of that class. This is really an elegant solution to pave our path towards the hybrid element case. And returning a const pointer secures the usage of shape object.

Implemented in ElementQ8, and ElementQ4.
