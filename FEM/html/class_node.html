void Node::setStrainAndStress ( const VectorXd &  strain,
const VectorXd &  stress 
)

Cumulate the stress and strain values at this node by summing up the values from each adjacent element.

Parameters
strainThe strain vector to be added to this node. The form is: [e(r), e(theta), e(z), gamma(rz)].
stressThe stress vector to be added to this node. The form is: [s(r), s(theta), s(z), t(rz)].
Note
This is an internal step for computing the averaged strain and stress at each node. The cumulative value will be averaged later.
